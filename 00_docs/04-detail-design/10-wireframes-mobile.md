## 4. Gesture Interactions Summary

### iOS Gestures
- **Pull to Refresh**: Pull down on folder/deck lists
- **Swipe Left**: Show action buttons (Edit, Delete)
- **Swipe Right**: Quick action (e.g., Quick Review)
- **Long Press**: Show context menu / Quick actions
- **Pinch**: Zoom (in card view with images)
- **Tap**: Select, navigate, flip card
- **Two-finger Swipe Left**: Undo
- **Swipe Down**: Dismiss modal, exit review

### Android Gestures
- **Pull to Refresh**: Pull down on lists
- **Long Press**: Show context menu
- **Tap Three-dot Menu**: Show options
- **Swipe**: Dismiss item, navigate
- **Pinch**: Zoom
- **Tap FAB**: Primary action
- **Swipe Up Bottom Sheet**: Expand modal

### Review Session Gestures
- **Tap Card**: Flip to reveal answer
- **Swipe Left**: Rate "Again" (hard fail)
- **Slight Swipe Left**: Rate "Hard"
- **Slight Swipe Right**: Rate "Good"
- **Swipe Right**: Rate "Easy" (perfect recall)
- **Swipe Down**: Exit review (with confirmation)
- **Double Tap**: Show card details

## 5. Animation Specifications

### Screen Transitions
- **Push Navigation**: Slide from right (iOS) / Fade + Slide (Android)
- **Modal Present**: Slide from bottom
- **Tab Switch**: Cross-fade
- **Duration**: 300ms (iOS), 225ms (Android Material)

### Card Animations
- **Flip**: 3D rotate Y-axis, 300ms
- **Swipe Out**: Slide + Fade, 200ms
- **Swipe In**: Slide + Fade, 250ms
- **Shuffle**: Stack cards with perspective

### Micro-interactions
- **Button Press**: Scale down 0.95, 100ms
- **List Item Select**: Highlight + Ripple (Android), Highlight (iOS)
- **Toggle Switch**: Slide, 200ms
- **Loading Spinner**: Rotate infinite
- **Skeleton Loader**: Shimmer left-to-right, 1500ms loop

### Success/Error Feedback
- **Success**: Toast slide down, stay 3s, slide up
- **Error**: Toast slide down with shake, stay until dismissed
- **Celebration**: Confetti particles on review complete

## 11. Accessibility Considerations

### Screen Readers
- All buttons labeled
- Card content read in order
- Form validation announced
- Progress announced in reviews

### Touch Targets
- Minimum 44x44 points (iOS)
- Minimum 48x48 dp (Android)
- Adequate spacing between targets

### Visual Accessibility
- WCAG AA contrast ratios
- Color not sole indicator
- Text resizable to 200%
- Support for large text

### Keyboard Navigation
- Tab order logical
- Focus indicators visible
- All actions keyboard accessible
