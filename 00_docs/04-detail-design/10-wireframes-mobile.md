# Mobile Wireframes - RepeatWise MVP

## 1. Screen Dimensions

### iPhone (375x812 - iPhone 13)
- Safe Area Top: 44px (status bar + notch)
- Safe Area Bottom: 34px (home indicator)
- Tab Bar Height: 83px (49px bar + 34px safe area)
- Navigation Bar: 44px

### Android (360x800 - Common)
- Status Bar: 24px
- Navigation Bar: 56px
- Bottom Navigation: 56px
- Floating Action Button: 56x56px

## 2. Navigation Structure

```
App Flow:
AuthStack → MainTabs → DetailScreens → Modals

├─ AuthStack
│  ├─ LoginScreen
│  └─ RegisterScreen
│
├─ MainTabs (Bottom Navigation)
│  ├─ HomeTab (FoldersScreen)
│  ├─ ReviewTab
│  ├─ StatsTab
│  └─ SettingsTab
│
├─ DetailScreens (Stack Navigation)
│  ├─ FolderDetailScreen
│  ├─ DeckDetailScreen
│  ├─ CardListScreen
│  └─ EditCardScreen
│
└─ Modal Screens (Fullscreen)
   ├─ ReviewSessionScreen
   ├─ ImportCardsScreen
   ├─ CreateFolderScreen
   ├─ CreateDeckScreen
   └─ ReviewCompletedScreen
```

## 4. Gesture Interactions Summary

### iOS Gestures
- **Pull to Refresh**: Pull down on folder/deck lists
- **Swipe Left**: Show action buttons (Edit, Delete)
- **Swipe Right**: Quick action (e.g., Quick Review)
- **Long Press**: Show context menu / Quick actions
- **Pinch**: Zoom (in card view with images)
- **Tap**: Select, navigate, flip card
- **Two-finger Swipe Left**: Undo
- **Swipe Down**: Dismiss modal, exit review

### Android Gestures
- **Pull to Refresh**: Pull down on lists
- **Long Press**: Show context menu
- **Tap Three-dot Menu**: Show options
- **Swipe**: Dismiss item, navigate
- **Pinch**: Zoom
- **Tap FAB**: Primary action
- **Swipe Up Bottom Sheet**: Expand modal

### Review Session Gestures
- **Tap Card**: Flip to reveal answer
- **Swipe Left**: Rate "Again" (hard fail)
- **Slight Swipe Left**: Rate "Hard"
- **Slight Swipe Right**: Rate "Good"
- **Swipe Right**: Rate "Easy" (perfect recall)
- **Swipe Down**: Exit review (with confirmation)
- **Double Tap**: Show card details

## 5. Animation Specifications

### Screen Transitions
- **Push Navigation**: Slide from right (iOS) / Fade + Slide (Android)
- **Modal Present**: Slide from bottom
- **Tab Switch**: Cross-fade
- **Duration**: 300ms (iOS), 225ms (Android Material)
