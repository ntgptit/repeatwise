## Responsive Breakpoints

### Desktop (> 1024px)

- Full sidebar + main content
- Multi-column layouts
- Hover states

### Tablet (768px - 1024px)

- Collapsible sidebar
- Stacked layouts
- Touch-friendly buttons

### Mobile (< 768px)

- Bottom navigation
- Full-width content
- Hamburger menu
- Stacked cards
